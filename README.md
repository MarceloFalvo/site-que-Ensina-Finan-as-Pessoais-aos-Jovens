# site-que-Ensina-Finan-as-Pessoais-aos-Jovens
O MF Invest é uma plataforma web criada para ajudar jovens de 16 a 30 anos a aprenderem educação financeira de forma leve, divertida e gamificada. O projeto foca em transformar conteúdos complexos sobre investimentos, organização financeira e planejamento pessoal em uma experiência visualmente envolvente, simples e prática.
